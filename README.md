# Clinical Trial SQL Analytics

Advanced SQL portfolio project simulating clinical trial data analytics for biopharma.

## Skills
- Complex JOINs
- CTEs
- Recursive CTEs
- Window Functions (LAG, LEAD, RANK, NTILE)
- Cohort Analysis
- Safety Monitoring
- Enrollment Analytics
- Query Optimization
- Views & Indexes
- Python + SQL integration

## Business Questions
1. Enrollment trends by site
2. Patient retention
3. Adverse event frequency
4. Treatment efficacy
5. Site performance ranking
6. Dropout prediction features
7. Protocol deviation analysis
8. Visit compliance
9. Rolling enrollment metrics
10. Clinical KPI dashboard
